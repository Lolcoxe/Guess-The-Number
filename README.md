# Guess The Number

A fun number guessing game with login/registration, score tracking, and leaderboard!

## Features
- 🔐 User authentication (login/register)
- 🎮 Guess numbers between 1-10
- 🏆 Leaderboard (top 5 players)
- 🌙 Dark mode toggle
- 💾 Persistent data with LocalStorage

## How to Play
1. Enter your name and password to create an account or login
2. Guess a number between 1 and 10
3. Get points for correct guesses
4. Check the leaderboard to see how you rank!

## Play Online
Visit: https://Lolcoxe.github.io/Guess-The-Number/

## Files
- `index.html` - Game interface
- `style.css` - Styling with dark mode support
- `script.js` - Game logic and account management
